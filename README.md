## Getting Started
Kick off your venv + install reqs

```
$ python3 -m venv venv
$ source venv/bin/activate
(venv) $ pip install requirements.txt
```

Now, if you want to change your output file, please feel free to do so in the run.py file 

```
$ python3 run.py
```
